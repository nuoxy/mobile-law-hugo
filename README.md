# Mobile-Law Hugo Site

A fast, mobile-first legal landing page built with Hugo.

## Features
-   **Mobile-First Design**: Optimized for small screens with a bento-grid layout.
-   **Speed**: Lightweight CSS, minimal JS, and static HTML generation.
-   **Security**: Content Security Policy (CSP) and no unsafe external links.
-   **Content**: Blog, FAQ, and Legal pages managed via Markdown.
-   **Sharing**: Native Web Share API integration for posts and deep-linked FAQs.

## Development

1.  **Install Hugo**:
    ```bash
    brew install hugo
    ```

2.  **Run Locally**:
    ```bash
    hugo server
    ```
    Visit `http://localhost:1313`.

## Structure
-   `content/`: Markdown content for pages, blog, and FAQ.
-   `layouts/`: HTML templates.
-   `static/`: CSS, JS, and images.
-   `hugo.yaml`: Site configuration.
